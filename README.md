# Перезагрузка — Telegram Mini App

Готовая вёрстка мини-аппа. Один файл `index.html`, без сборки и зависимостей.

## Что уже сделано под Telegram
- Подключён официальный SDK `telegram-web-app.js`
- Цвета берутся из темы Telegram (светлая/тёмная подхватываются автоматически)
- Сохранение через **CloudStorage** — прогресс синхронизируется между устройствами пользователя (с fallback на localStorage, если открыть вне Telegram)
- Кнопка «😶 Мне сейчас пусто» вынесена в нативную **MainButton** внизу экрана
- Тактильная отдача (HapticFeedback) на нажатия
- `expand()` на весь экран, поддержка safe-area

## Как запустить (3 шага)
1. **Захостить файл по HTTPS.** Telegram требует https. Самое быстрое:
   - GitHub Pages: положить `index.html` в репозиторий → Settings → Pages → включить. Получишь ссылку вида `https://username.github.io/repo/`.
   - Либо Netlify / Vercel / Cloudflare Pages — перетащить папку, получить ссылку.
2. **Создать бота** у [@BotFather](https://t.me/BotFather): `/newbot` → имя → токен.
3. **Привязать мини-апп:**
   - В BotFather: `/newapp` → выбрать бота → задать название, описание, иконку → вставить URL из шага 1.
   - Или быстрее — `/setmenubutton` → выбрать бота → вставить URL → задать текст кнопки (напр. «Открыть трекер»).

После этого кнопка меню в чате с ботом открывает приложение.

## Локальная проверка
Открыть `index.html` в браузере — работает и так (через localStorage и плавающую кнопку вместо MainButton). Но CloudStorage и тема Telegram появятся только внутри клиента Telegram.

## Если захочешь развить
- Напоминания (уведомления от бота утром) — нужен серверный бот, шлющий сообщения по расписанию.
- Бэкап прогресса на свой сервер — сейчас данные живут в CloudStorage Telegram (до 4 КБ на ключ, нам хватает).
